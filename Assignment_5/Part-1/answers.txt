Q1 Explain r,w,x?
-> r- for reading a file we use r as a permissions 
-> w- for writing a file we use w as a permissions
-> x- for executing a file we use x as a permissions

Q2 What happens if permissions are set to 777?
-> If we set permsissions to 777 we give all the permisions to users,groups,ad others.

Q3 How does permissions help in securing data ?
-> We cannot give all permissions to users and groups and other so we set permisions via chmod command to keep our whole system secure.

Q4 what does x field indicate ?

Q6 why should root ssh login be disabled?
-> Because allowing root ssh login gives attackers a direct permission or login details and can allow brute force easily.
